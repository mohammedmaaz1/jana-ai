# 🌐 Jana AI Platform — Landing Website

## 🚀 Overview

This project is a modern, responsive landing website for the **Jana AI Platform**, built to present a vision of transforming fragmented daily services into seamless, outcome-driven experiences.

The platform combines **AI intelligence**, **human support (JanMitra)**, and **case orchestration** to simplify real-world service delivery at scale.

---

## ✨ Key Idea

> **“From Services to Outcomes”**

Instead of handling isolated tasks, Jana AI focuses on delivering **complete outcomes**, supported by both intelligent systems and human continuity.

---

## 🧩 Core Concepts Highlighted

* **People-First AI**
  AI enhances human capabilities rather than replacing them.

* **JanMitra Support Layer**
  Every user case is supported by a dedicated human associate ensuring continuity and trust.

* **Case-Oriented Approach**
  Needs are converted into structured workflows that are executed step-by-step until completion.

* **Full Ecosystem Execution**
  Delivered through apps, service providers, and intelligent tools.

## ⚙️ Getting Started (Local Setup)

### 1. Clone the repository

```
git clone https://github.com/mohammedmaaz1/jana-ai.git
cd jana-ai
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm run dev
```

Visit:

```
http://localhost:3000
```
